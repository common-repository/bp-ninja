Site admins can use stealth mode to hide their last activity.

== Description ==

Site admins can use stealth mode so other users cannot see their last activity.
Stealth mode enable/disable is in the admin bar.

== Installation ==

1. Upload `bp-ninja` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
